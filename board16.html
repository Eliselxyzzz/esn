SRIO-VX213 - Ethernet Switch VPX Data Exchange Board

SRIO-VX213 - Ethernet Switch VPX Networking Module is designed and developed based on the standard 6U OpenVPX system structure, with Serial Rapid IO Switch and unmanaged Ethernet Switch card. Fundamentally the Boards meet OpenVPX standard platform, where it is widely used in large scale processing of Radar Imaging signal, Electronic Signal Investigation, Image processing, Sonar, Communications/Command/Control Exchanged Hub.
  1. The Key Features and Characteristics
    2.
    1. 5 CPS1848 onboard chip, a VSC7460 and an FPGA, meet OpenVPX standard MOD6-SWH-20U19F-12.4.1-n defined
    2. Rapid IO ID configuration flexible, diverse, simple, support dynamic configuration and static configuration
    3. Ethernet configuration method is flexible, diverse and simple
    4. High reliability hardware board design, provides third-party environmental test report

  2. The Overall Technical Specifications
    3.
    1. Rapid IO switching capacity
      • Board provides 18 through the VPX connector to the backplane x4 Gen2 Serial RapidIO Ports
      • Front panel provides two QSFP Ports realization of foreign Serial RapidIO Switch
    2. Ethernet switching capabilities
      • Board provides 20 through the VPX connector to the backplane Ethernet 1G Serdes Ports
      • Front panel provides two RJ45 interface to connect to the Ethernet Switch

  3. The Mechanical/Physical Specification
    4.
    1. Size: Standard 6U boards
    2. Operating temperature: - 40 ° C ~ 70 ° C
    3. Power consumption: about 70W
    4. Heat: conduction-cooled cooling and air cooling

  4. The External Interface
    5.
    1. Provide three (3) x 10/100 / 1000 BaseT Ethernet port from front panel, one connected to the FPGA, 2 connections to the Ethernet switch chip
    2. 1 QSFP interface or a Fiber optic interface that provides a 4 X Rapid IO interfaces
    3. 8 LED indicators
