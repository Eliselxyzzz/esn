HSSP-V7-66678 High Speed Signal Processing board

HSSP-V7-66678 High Speed Signal Processing Module is designed based on TI Multi core TMS320C6678 DSP universal processing board and with OpenVPX platform to meet the standard, where applicable to Radar Imaging Signal Processing, Electronic Signal Investigation, Navigation, Guidance, Remote sensing, Weather Information and Ground Imaging Processing, Sonar Signal Analysis and other high-speed real-time signal processing tasks for reliable, resilience hardware protection.
  1. The characteristics of the board
    2.
    1. The board contains 4 C6678 chips and 1 K7FPGA, for the industry's largest processing capacity of a single board
    2. Reliable, stable, flexible drive library, algorithm library, users quickly get started
    3. High reliability hardware board design, provide third party environment test report
    4. Designed to be customized according to the actual needs of users

  2. General Board Technical Specification
    3.
    1. Processing capacity
      • 4 x C6678 processing capacity: 640GFLOPS
      • 1 x XC7K325T processing power: 840 Slices DSP48E1

    2. Storage capacity
      • DSP plug-in DDR3 total capacity: 8GB/16GB/32GB
      • FPGA plug-in DDR3 total capacity: 1GB

    3. Rapid IO transmission bandwidth
      • 710MB/S@2.5Gbps
      • 920MB/S@3.125Gbps
      • 1400MB/S@5Gbp

    4. HyperLink transmission bandwidth
      • 1400MB/S@5Gbps
      • 1850MB/S@6.25Gbps

    5. External Interface
      • Net mouth
      • QSFP-B optical Fiber interface
      • JTAG-C interface

    6. Physical Specification
      • Size: Standard 6U board
      • Operating temperature: - 40 C ~ 70 degrees C
      • Storage temperature: -55 ~ 85 C ~
      • Power consumption: about 72W
      • Cooling: Force cooling and normal cooling
