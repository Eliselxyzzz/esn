The Fundamental Board Technical Specification

  1. Core Processing System
    2.

    2. Integrated 1 Artix-7 Xilinx series FPGA (optional configuration)

  2. Storage Capacity
    3.

    2. DSP 128Mb SPI NOR FLASH, for Boot DSP

  3. The Main Interface Interconnection Specification
    4.
    1. DSP 4x through 1 sets of HyperLink a high-speed interface between the Internet, the highest rate of line up to 6.25Gbps

    3. DSP 2x and FPGA through 1 sets of SRIO C interface
    4. DSP and SPI between FPGA and GPIO interface
    5. External interface for the PCIE 2x, can be configured to select from the FPGA or DSP-0
    6. External interface can be configured as a 2x or GTX 4x interface (4x mode can not be used PCIE FPGA function
    7. SGMII external 2 GbE g interface, respectively, by the two DSP implementation
